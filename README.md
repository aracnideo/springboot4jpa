# SpringBoot4JPA
This project was developed as part of my studies in Spring Boot and JPA/Hibernate.
It demonstrates a REST API with basic CRUD operations using Spring Boot, following layered architecture and best practices.

## Project Goals
- Practice Spring Boot application structure
- Implement RESTful endpoints
- Use JPA/Hibernate for ORM
- Apply layered architecture (Controller + Service + Repository)
- Handle exceptions properly
- Understand entity relationships

## Technologies Used
- Java 25
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 / SQL Database
- Git & GitHub

## Features
- Create resources
- Update existing resources
- Delete resources
- Find by ID
- List all records

## How to Run

1. Clone the repository
2. Open the project with your preferred IDE
3. Run the Spring Boot application
4. Access the API using Postman or a browser

## Concepts Practiced
- REST API development
- Spring Boot framework
- JPA/Hibernate ORM
- Entity relationships
- Exception handling
- Layered architecture
